CARTA

int idcarta | String marca | String modelo | int motor | int cilindros | int potencia | int revoluciones | int velocidad | double consumo

- Obtener todas las cartas

GET : http://localhost:8080/ProyectoCartas/api/carta/cartas

- Obtener carta por {id}

GET : http://localhost:8080/ProyectoCartas/api/carta/{id}

- Crear carta (atributos arriba ↑ )

POST : http://localhost:8080/ProyectoCartas/api/carta/crear

- Editar carta por {id}

PUT : http://localhost:8080/ProyectoCartas/api/carta/{id}

- Eliminar carta por {id}

DELETE : http://localhost:8080/ProyectoCartas/api/carta/{id}

- Obtener ultima carta creada (como ampliacion)

GET : http://localhost:8080/ProyectoCartas/api/carta/last
